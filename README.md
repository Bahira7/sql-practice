# SQL Practice — World Population Analysis

Learning SQL for data engineering using a world population dataset (233 countries).

## Dataset
- Source: Kaggle — World Population Dataset by Sourav Banerjee
- 233 rows, 17 columns
- Loaded into PostgreSQL via DBeaver

## Structure
- `phase1_queries.sql` — Foundations: SELECT, WHERE, GROUP BY, aggregations, filtering

## Topics Covered
- SELECT, FROM, WHERE
- AND, OR, NOT, IN, BETWEEN, LIKE, IS NULL
- ORDER BY, LIMIT
- COUNT, SUM, AVG, MIN, MAX
- GROUP BY, HAVING
- Aliases with AS

## Tools
- PostgreSQL
- DBeaver
